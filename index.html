Why feathers

feathers is a starting point for your next Angularjs project. Here we'd like to share a lot of best practices on coding with AngularJS with you.

feathers use the following open sources projects:

  • AngularJS AngularJS lets you write client-side web applications as if you had a smarter browser.
  • Angular-ui Router The de-facto solution to flexible routing with nested views
  • karma A simple tool that allows you to execute JavaScript code in multiple real browsers.
  • protractor Protractor is an end-to-end test framework for AngularJS applications
  • mocha Mocha is a simple, flexible, fun JavaScript test framework for node.js and the browser. For more information view the documentation.
  • chai Chai is a BDD / TDD assertion library for node and the browser that can be delightfully paired with any javascript testing framework.
  • sinon Standalone and test framework agnostic JavaScript test spies, stubs and mocks.

Moreover, with these cool things.

  • jade
  • node-sass
  • coffeescript

Getting Start

Quick Start

$ git clone https://github.com/Wiredcraft/feathers.git
$ cd feathers
$ npm install && bower install
$ gulp dev

Development

First of all, you should start an dev server.

$ gulp dev

Tests

Unit Tests

libs:

  • karma
  • mocha
$ karma start

e2e Tests

libs:

  • protractor
  • mocha
$ webdriver-manager start
$ gulp dev
$ gulp test:e2e

Build Documentation

  • metalsmith
$ cd docs
$ npm install
$ node metalsmith.js
$ node web-server.js

Now go to http://localhost:3000.

Deploy Document

$ cd ..
$ pwd
~/workspace/feathers
$ git add -f docs/build && git commit -m "commit for gh-pages"
$ git sutree push --prefix docs/build origin gh-pages
