## Why feathers?

`Feathers` wants to share a lot of best
practices on coding with AngularJS with you.

## Quick Start

Install Node.js and then:

```sh
$ git clone git://github.com/Wiredcraft/feathers
$ cd feathers
$ npm install
$ bower install
```

Happy hacking!
